# FinOps

## Cloud Computing

The layer underneath everything else. Before Kubernetes, before the service mesh, before the agents, there is a bill and someone has to explain it.

This section holds what does not belong to the cloud-native or AI-native arcs but underpins both: hyperscaler architecture (Azure, AWS), and cloud financial management. FinOps sits here rather than in a governance section, because in practice it is not a governance discipline. It is an operating model, and it fails for cultural reasons far more often than technical ones.
