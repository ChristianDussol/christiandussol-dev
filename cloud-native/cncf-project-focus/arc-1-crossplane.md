# Episode 2: Crossplane

## **One Kubernetes API. Any Cloud Provider**

Crossplane is the most underrated CNCF project in the platform engineering space. This episode walks through the four core components, the migration path from Terraform and why Compositions are the right abstraction for a platform contract.

* Providers, Managed Resources, Composite Resources and Compositions
